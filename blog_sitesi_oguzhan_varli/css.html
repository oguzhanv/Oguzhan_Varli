CSS

Merhaba,

Bugün CSS'in önemli bir özelliği olan flex kavramından bahsedeceğiz.
Bildiğiniz üzere HTML elementlerinin block ve inline olmak üzere iki çeşit türleri vardır.

Yaygın Block Elementleri: Div, Header, P, Table, Ul, H1...H6, Section... gibi elementlerdir.

Yaygın Inline Elementleri: A, Button, Input, I, Img, Label, Span... gibi elementlerdir.

Block Elementler aşağıdaki resimdeki gibi sayfanın bütün genişliğini kendi yüksekliği kadar kullanmaktadır.



Block Elementler ardından gelen elementin türüne bakmaksızın gelen elementi kendi yüksekliğinin altına yerleştirir.



Buradaki ikinci elementin alta geçmesinin sağlayan olay block elementinin display özelliğinin block olmasıdır.

Inline etiketlerse block elementlerin aksine satırı kaplamak yerine, aynı satıra bir veya birden fazla inline
element(ler) alabilirler.

(Block elementler satırı tamamen kapladıkları için öcesindeki element inline olsa bile yanına yerleşmezler.)



CSS'in display flex özelliği ( display: flex; ) ise bulunduğu kapsayıcının(container) içindeki block elementlerin
block özelliklerini kaybedip inline elementlere dönüşmesini sağlar.Böylelikle birden fazla block veya inline elementi
aynı satırda sıralayabiliriz.


Kapsayıcıya display flex özelliği verildikten sonra;


Böylelikle;
Elementleri rahatlıkla konumlandırabiliriz.(Merkezde ortalamak, kolon olarak konumlandırmak gibi..)

Elementleri aynı satırda hizzalamak için isteğe göre aşağıdaki komutlar kullanılabilir.



Justify-content: center             ==> Elementleri aynı satır içerisinde merkezde hizalar.
Justify-content: space-between      ==> Elementleri aynı satır içerisinde başa ve sona bir tane konumlandırarak kalanı eşit hizalar.
Justify-content: space-evenly       ==> Elementleri aynı satır içerisnde aynı boşlukla hizalar.
Flex-direction: column              ==> Elementleri aşağıdan yukarıya doğru aynı hizada sıralar.
